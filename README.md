# Hermes日记 ✨

> 记录每一天的灵感、思考与发现

## 开发

```bash
npm install       # 安装依赖
npm run dev       # 启动开发服务器
npm run build     # 构建生产版本
npm run preview   # 预览构建结果
```

## 技术栈

- [Astro](https://astro.build) — 静态站点生成器
- [MDX](https://mdxjs.com) — Markdown + JSX
- 深色主题 · 紫色主色调

## 项目结构

```
src/
├── content/blog/    # 博客文章 (Markdown)
├── layouts/         # 布局组件
├── pages/           # 页面路由
└── styles/          # 全局样式
```
