## 一张黑白头像，四种风格

这是 Hermes Agent 最近完成的一次工作流实验——用户提供一张黑白线稿头像，Hermes 自动调用 nanobanana 图像生成服务，输出四种截然不同风格的彩色版本。

![黑白线稿原图](/images/hermes_agent_logo.jpg)

Hermes 调用服务后生成的四种风格：

![四种风格上色结果](/images/hermes-nanobanana-colorize.jpg)

四格图从左上角顺时针分别是：

- 🟣 **香港霓虹风**——深紫色发色，背景是夜晚的霓虹招牌，街头气息扑面而来
- ☕ **复古茶餐厅风**——棕色发色，背景是港式茶餐厅的马赛克墙砖和一杯热奶茶
- 📻 **黑胶唱片店风**——深棕色发色，背景是满墙的黑胶唱片和老式磁带机
- 🌿 **新古典城市风**——墨绿色发色，背景是几何花窗与远处的城市线条

同一张脸，四种时间与空间的投影。

## 工作流是怎么运作的

整个流程从一句提示词开始：

> 帮我把这张黑白头像做成四种不同风格的彩色版本，分别是香港霓虹、复古茶餐厅、城市新古典、黑胶唱片店风格。

Hermes 拿到请求后，完成了以下步骤：

### 1. 解析任务，拆解参数

Hermes 首先把"四种风格"拆解成四组独立的生成参数，每组包含：
- 发色（hair color）
- 背景场景（background scene）
- 配色基调（color palette）
- 情绪关键词（mood keywords）

这个拆解步骤是关键。大部分用户的需求都是模糊的，Hermes 的工作是把模糊需求翻译成可执行的结构化参数。

### 2. 调用 nanobanana 生图

nanobanana 是一个专注于二次元风格图像生成的服务，对于动漫风格的头像上色有很强的一致性保持能力——也就是说，在改变风格的同时，人物的基础特征（脸型、发型结构、耳机细节）会保持稳定。

Hermes 通过工具调用（tool call）的方式把四组参数依次发送给 nanobanana，拿回四张独立生成的图片。

### 3. 拼合输出

最后 Hermes 把四张图片拼合成一张 2×2 的对比图，方便用户直观比较风格差异。

## 这个工作流的价值在哪里

用一句话概括：**它把"想象力"转化成了"可复现的流程"。**

传统做法是：用户自己去找AI绘图工具，手动调参，一个风格一个风格地试，遇到不满意的再调，来回迭代。这个过程耗时且需要一定的提示词经验。

Hermes 的做法是：用户只需要说"我想要这四种风格"，其余的参数设计、接口调用、多图拼合都由 Agent 完成。用户只需要在最后判断结果是否满意。

这不是单纯的"自动化"，而是**把专业知识（风格参数、接口调用、图片处理）封装进 Agent 的工具链**，降低了用户的认知门槛。

## 可以拓展的方向

这套工作流目前还只是一个原型，可以延伸的方向有很多：

- **批量处理**：一次输入多张头像，统一生成某种风格的系列图
- **风格学习**：让 Hermes 从用户反馈中学习哪种风格组合最受偏爱
- **自定义风格包**：用户自定义风格参数，保存为可复用的"风格模板"
- **动态调整**：基于图片的反馈（如"这张太暗了"），Hermes自动修正参数重新生成

本质上，这个实验验证了一件事：**Hermes Agent 不只是一个对话工具，它是一个可以驱动专业创作的工作流引擎。**
